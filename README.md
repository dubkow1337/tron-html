## 🎯 ОБ ИГРЕ

**Tron: Light Cycles** — классическая аркада в стиле вселенной «Трон: Наследие».  
Два мотоцикла оставляют за собой неоновые линии-стены. Побеждает тот, кто дольше продержится, не врезавшись в стену, границу или собственный след.

Игра выполнена в киберпанк-стиле с пульсирующим свечением, динамической камерой, частицами и голосовым сопровождением (Microsoft Pavel).

---

## ✨ ОСОБЕННОСТИ

| Фича | Описание |
|------|----------|
| 🎮 **2 игрока** | Классический режим: синий (стрелки) против оранжевого (WASD) |
| 🤖 **VS AI** | Умный противник, который анализирует 30 шагов вперёд |
| ☠️ **Выживание** | Бесконечные красные враги появляются каждые 2 секунды |
| 🏆 **Турнир** | Серия матчей до 3 побед, объявление чемпиона |
| ⚡ **Ускорение** | Скорость растёт каждые 200 шагов |
| 💥 **Частицы** | Искры за мотоциклами |
| 🔊 **Голос** | Обратный отсчёт и объявление победителя (мужской, русский) |
| 🎨 **Blur-эффект** | Чем быстрее едешь, тем сильнее размытие |
| 🏆 **Рекорды** | Лучший результат сохраняется в браузере |

## 🎮 УПРАВЛЕНИЕ

| Действие | Синий (🔵) | Оранжевый (🟠) |
|----------|------------|----------------|
| **Вверх** | `↑` | `W` |
| **Вниз** | `↓` | `S` |
| **Влево** | `←` | `A` |
| **Вправо** | `→` | `D` |
| **Пауза** | `ESC` | `ESC` |
